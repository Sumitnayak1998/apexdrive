# ApexDrive - Premium Car Rental Application

![ApexDrive Logo / Banner Placeholder](https://via.placeholder.com/1000x300?text=ApexDrive+Car+Rental)

ApexDrive is a modern, high-end car rental web application designed to provide users with a seamless, premium vehicle booking experience. The platform moves away from traditional web forms by utilizing an innovative, automated **Telegram Bot Booking System**, allowing users to secure their vehicles instantly via chat.

## ✨ Key Features

- **Cinematic Premium UI**: A highly polished, modern aesthetic utilizing glassmorphism, deep cinematic color grading, and smooth scroll animations.
- **Telegram AI Booking Integration**: Users browse the fleet and are seamlessly redirected to a Telegram bot (`@car_rental_helper_bot`) to complete their booking, creating a frictionless, mobile-first experience.
- **Dynamic Fleet Showcase**: A compact, responsive grid displaying 20 real-world vehicles (Sedans, SUVs, Luxury, and EVs) with accurate pricing, specifications, and Wikipedia-sourced imagery.
- **Responsive Design**: Flawlessly adapts to any screen size, ensuring the premium feel is maintained from desktop monitors to mobile phones.
- **Micro-interactions**: Enhanced user engagement through subtle hover states, glowing elements, and `framer-motion` entrance animations.

## 🛠️ Tech Stack

- **Frontend Framework**: React.js
- **Styling**: Tailwind CSS (Utility-first CSS)
- **Animations**: Framer Motion (Declarative animations)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 💻 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumitnayak1998/apexdrive.git
   cd apexdrive
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open `http://localhost:5173` (or the port provided in your terminal).
